## 정규표현식

### 1. [] : 문자클래스

- [] 사이의문자들과 매치 ( 대괄호 안에는 어떤 문자도 가)
- ex) 정규식 : [abc]
- "a" : 매칭
- "begin" : 매칭
- "test" : 매칭 x 

ex) 

[0-7] : 0에서 7까지( [ 01234567] )

[a-z] : a에서 7까지( [ 01234567] )

[a-zA-Z]

[^0-9] : 숫자가 아닌 문자 

\d : 숫자 = [0-9]  (의미동일)

\D : 숫자가 아닌것 : 매칭

\s : [\t\n\r\f] : 공백에 해당함 

\S : [ ^\t\n\r\f ]

######### 대문자를 쓰면 소문자 표현식과 의미 반대

\w : 문자 + 숫자,  [a-zA-Z0-9]

. (점) : 모든문자(\n 문자 제외)

ex) 정규식 : a.b 라고 정의했다고 가정

akb : 매칭

a5b : 매칭

aby : 매칭 x  (a b 사이에 문자가 와야한다 )

정규식 : a[.]b 과 a.b는 완전히 다른 의미
