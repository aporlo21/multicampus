**2.2 조건문 ( if )**

 c = 15 * 5
>>> d = 15 + 15 + 15 + 15 + 15
>>> if c > d:                      # 만약 c가 d보다 크면
>>> ...     print('c > d')             # 'c > d'라고 출력한다.
>>> ... elif c == d:                   # 그렇지 않고 c와 d가 같다면
>>> ...     print('c == d')            # 'c == d'라고 출력한다.
>>> ... else:                          # 이도 저도 아니면
>>> ...     print('c < d')             # 'c < d'라고 출력한다.
>>>
>>>
>>>
>>> c == d

* elif : 여러 개의 조건을 검사해서 그 중에서 맘에 드는 것을 고를수 있다.
* c == d : c & d의 값이 같다.
